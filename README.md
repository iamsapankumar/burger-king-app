# BURGER KING BR APP CLONE

Burger King App with react-native and expo.

---

### Installation

```sh
yarn install
```

---

### Setup

Change file `./src/services/api.js` with your ip address

---

### Run

Server run mock with json-server

```sh
yarn server
```

Run app

```sh
yarn start
```
